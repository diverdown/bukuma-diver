# ブクマダイバー
面白いページを1匹見つけたら裏に100匹潜んでいると考えろ！良質なコンテンツを芋づる式に発掘していく装置、ブクマダイバー！

## Development dependency
- rbenv
- nvm
- chromedriver

## Setup for develpment
```sh
$ git clone https://github.com/webken/bukuma-diver.git
$ cd bukuma-diver
$ npm install -g bower gulp coffee-script
$ npm install
$ bower install
$ bundle
```

## Develpment
```sh
foreman start
```
